# Blockchain-Based E-Voting System

A secure and transparent electronic voting system built using Blockchain technology. The system ensures vote integrity, prevents tampering, and provides a decentralized approach to election management.

## Features
- Secure voter authentication
- Election and candidate management
- Blockchain-based vote storage
- One voter, one vote mechanism
- Transparent and tamper-proof voting
- Automatic result calculation

## Tech Stack
- React.js
- Node.js
- Express.js
- Hyperledger Fabric
- MySQL

## Objective
To provide a secure, transparent, and decentralized voting platform that guarantees election integrity while maintaining voter privacy.
